üìñ How Guide Parsing Works

Header Format

  • First non-empty line: Guide Title
  • Following lines until blank: Version/Description (optional)

Section Format

  • Non-indented text: Section headers
  • Lines after header (before steps): Section description
  • A blank line: Start a new section

Step Format

  • Indented lines or lines starting with -: Steps within a section
  • More indentation: Sub-steps (nested)

Example
